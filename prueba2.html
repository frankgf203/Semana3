Prueba N°2 - Operador prompt en JavaScript

El operador prompt en JavaScript se utiliza para mostrar un cuadro de diálogo que solicita al usuario que ingrese información. Esta función devuelve el valor ingresado por el usuario como una cadena de texto, permitiendo la interacción dinámica con la página web.
