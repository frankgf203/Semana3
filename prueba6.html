Prueba N°6 - Uso de For en JavaScript

En JavaScript, el bucle `for` permite ejecutar un bloque de código un número específico de veces, utilizando una estructura compacta para inicializar, evaluar y actualizar la variable de control. Es ideal para iteraciones con un conteo definido y predecible.
